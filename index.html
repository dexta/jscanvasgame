Jscanvasgame

little game framework with javascript and canvas


Welcome to js canvas game.

this framework should be simple, tiny and modular, one file is enough to start your first mock-up. Just one dependency to jQuery, do so mutch in the DOM Tree to show up your information as you can. Canvas can be very slow in some Browser Implemetation and cases, but have the power of geometry.


the framework starts with the arcade game Pong

Pong is one of the earliest arcade video games; it is a tennis sports game featuring simple two-dimensional graphics. The Name was a hint to worlds first action electric game. In 1958, William Higinbotham created a game called Tennis for Two, a competitive two-player tennis game played on an oscilloscope. The players would select the angle at which to put their racket, and pressed a button to return it. Although this game was incredibly simple, it demonstrated how an action game (rather than previous puzzles) could be played on a computer.[2]

First idea was be able to start with a basic setup.

  • Canvas DOM to JavaScript, get Size or define it
  • Key Events sets and configurable
  • Mouse Buttons Event and moves stored as Object
  • Touch to Mouse (just one finger as yet)
  • simple FPS counter, to prove the performance

use the javascript objects to store the data

// define the ball
var bPos = {x:50,y:50,w:10,h:10};
// define a direction object
var bDir = {x:-1,y:1};
...
function draw() {
  bPos.x += bDir.x*6;
  bPos.y += bDir.y*4;
  ...
  // test if the ball is outside the view
  if(bPos.y<0 || (bPos.y+bPos.h)>HEIGHT) {
    // if lower then 0 or greater than height -> change the direction
    bDir.y *= -1;
    }
  if(bPos.x<0 || (bPos.x+bPos.w)>WIDTH) { 
    bDir.x *= -1;
    }
}

These lines of code draw a rectangle in the canvas, move it and test the collision with the edge.
